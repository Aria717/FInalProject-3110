# Installation Guide

0. **Update OPAM**
    ```bash
    $ opam update
    $ opam upgrade
    $ opam install ansiterminal 
    ```

1. **Clone the Repository:**
   ```bash
   git clone git@github.com:Aria717/FinalProject-3110.git
   ```

2. **Navigate to the Directory:**
   ```bash
   cd chess
   ```

3. **Dune execute:**
   ```bash
   dune exec bin/main.exe
   ```
